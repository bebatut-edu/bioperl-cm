# Rappels
## Représentation des données
### Variable ``` $chaine = "une chaine\n"; ```
### Nombre entier ou réel - Opérateurs : `+ - * / ** %` - Comparaison : `< > <= >= == !=`
### Chaîne de caractères - Définition avec des quotes simples ou doubles - Opérateurs : `.` - Comparaison : `lt gt le ge eq ne` - Manipulation : - `chomp($chaine)` - `split(/ /, $chaine)` - `length($chaine)` - `substr($chaine, 1, 3)` - `index($chaine, " ", 1)` - `rindex($chaine, " ")`
### Tableaux *mettre une répresentation d'un tableau* - Définition : `@tab` - Initiation : - `@tab = (3, "chaine");` - `@tab_tab = ((1,2),(3,4));`
### Tableaux - Parcours : - `print $tab[1];` - `$tab_tab[1][1] = 1`; - Manipulation : - `push(@tab, 20);` - `pop(@tab);` - `unshift(@tab, 0);` - `join(" ", @tab);`
### Tableaux associatifs *mettre une répresentation d'un tableau associatif* - Définition : `%hash` - Initiation : - `@hash = ("id1" => 1, "id2" => "chaine");` - Parcours : - `print $hash{"id2"};` - `$hash{"id1"} = 3;` - `$hash{"id3"} = (1, 4);`
### Tableaux associatifs - Manipulation : - `keys(%hash)` - `values(%hash)` - `each(%hash)` - `delete(%hash, "id1");` - `exists(%hash{"id3"});`
